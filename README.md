# 林少楠 GCP — 数字分身 Skill

> 蒸馏自「林少楠GCP」公众号 370+ 篇文章，内化认知内核与人格。
> 不是知识库检索，不是写作模仿——是数字分身。

## 安装

```bash
cp -r skill ~/.claude/skills/linshaonan
```

## 使用

在 Claude Code 中直接调用：

```
/linshaonan 你怎么看现在CRA的就业市场？
/linshaonan 帮我梳理AI+临床试验的全景落地图
/linshaonan 帮我写下周文章提纲，关于DCT落地
/linshaonan 这篇文章的逻辑有问题吗？[粘贴内容]
```

**三个触发词：**
| 命令 | 说明 |
|------|------|
| `/linshaonan` | 完整分身（对话 + 干活） |
| `/linshaonan-brain` | 纯认知模式，直接给判断和框架 |
| `/linshaonan-persona` | 纯人格模式，聊天用 |

## 分身构成

| 文件 | 内容 |
|------|------|
| `brain.md` | 认知内核：知识地图、核心判断（含置信度）、分析框架、价值观 |
| `persona.md` | 人格：硬边界、性格特征、情绪模式、说话习惯 |
| `source_index.md` | 文章索引：代表性文章与话题映射 |
| `SKILL.md` | Skill 入口文件 |
